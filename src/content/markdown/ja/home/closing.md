## 私たちの物語

Rezics の Re は reincarnation に由来します。神秘的な意味ではなく、もっと単純な考えです。人が創り、学び、愛したものは本当には消えず、新しい言語、メディア、コミュニティ、理解の中で再び現れます。

私たちは最初に Web 小説とデジタルネイティブな書籍から始めました。そこでは、プラットフォーム、言語、版、tag、wiki、コミュニティ文脈の分断が早くから明らかだったからです。しかし Rezics が向き合うのは本だけではありません。保存され、理解され、再び出会われるべきすべての物語です。

それが inherited · create · spread です。すでにある知識を受け継ぎ、新しい理解と創作を生み、愛されるものを広げ続けることです。
